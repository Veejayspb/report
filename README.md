Отчёты
============

Генератор отчетов о проделанной работе на основе активности в указанных GIT репозиториях.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Установка
------------

```sh
cd path/to/dir
git clone git@github.com:Veejayspb/report.git
composer install
```
Создаем в корне проекта файл конфигурации **config.php** из **config.php.dist** и заполняем его, согласно инструкциям внутри.
